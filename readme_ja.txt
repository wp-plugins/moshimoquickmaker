***********************************************
概要
***********************************************
このプラグインは、日本のドロップシッピングサービスの"もしも"をより簡単に運用するために開発されたものです。プラグインで任意のカテゴリーを選択すると、商品はあなたのwordpress内に自動的にインストールされます。しかも、定期的に自動で新しい商品に更新します。多くの人は"ドロップシッピングは興味あるが、手間がかかる"と言います。その要望に応えるために作りました。弊社サイトでテンプレートや拡張ウィジェットを有料で販売しています。購入して今後の開発にご協力ください。

***********************************************
ご注意
***********************************************

1.本プラグインを使用するには「もしも」のアカウントが必要になります。
2.本プラグインは「もしも API」より商品データをあなたのサーバーに定期的自動ダウンロードします。
3.本プラグインは設定画面、テーマ（ページ内）にお知らせや広告を表示致します。
4.「もしもAPI」の仕様変更があった場合に、本プラグインは正常に動作しなくなる事があります。

***********************************************
使用方法
***********************************************

1. "quickmaker"フォルダーをWordpressの"/wp-content/plugins/"内にフォルダーごとコピーします。
2. WordPressの管理画面プラグインメニューより"MoshimoQuickMaker"を有効化してください。
3. WordPressの管理画面のテーマより'MoshimoQuickMakerTemplate01'を有効化してください。
4. プラグインが有効化されると管理画面に"QuickMaker"設定メニューが表示されますので、各初期設定を入力し下部の保存ボタンで保存してください。
5. 同"QuickMaker"カテゴリーメニューから任意のカテゴリーを選択して下部の設定ボタンで決定してください。


詳しくは下記アドレスをご参照ください。
(http://www.a-ings.net/user_data/quickmaker.php)
